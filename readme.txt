== Installation ==

It's really easy and super simple to install **Next FeedBuilder** plugin but before installing **Next FeedBuilder** make sure you've installed [Elementor](https://wordpress.org/plugins/elementor/ "Install Elementor"). **Elementor** or **Gutenberg** is the only dependency.

= Automatic Installation =
1.  Go to `Plugins > Add New` screen in WordPress.
2.  Search for `Next FeedBuilder`.
3.  Install and activate the plugin, that's it.

= Manual Installation =
1.  Download [Next Addons](https://downloads.wordpress.org/plugin/next-feed-builder.zip "Download Next FeedBuilder").
2.  Extract the `next-feed-builder.zip` file. You'll get plugin files inside `next-feed-builder` directory.
3.  Upload the plugin files to the `/wp-content/plugins/next-feed-builder` directory.
4.  Activate the plugin through the 'Plugins' screen in WordPress.
